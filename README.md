# BDE Gummy — Design Reference Repository

This repo contains all reference materials for the BDE Gummy landing page redesign.

## Folder Structure

```
/references
  /ref-1-gruns-olipop     → Grüns x OLIPOP reference screenshots
  /ref-2-khloud           → Khloud Protein Popcorn reference screenshots
/specs
  /business-sheet.md     → BDE Gummy Business Specification (B1–B11)
  /technical-sheet.md    → BDE Gummy Technical Specification (T1–T6)
  /composite-brief.md    → Combined page brief for Stitch / v0.dev / Figma
```

## How to Use With Google Stitch
1. Go to [stitch.withgoogle.com](https://stitch.withgoogle.com)
2. Import this repo
3. Point Stitch to `/references` for visual inspiration
4. Use `/specs/composite-brief.md` as the prompt

## Stack
- Next.js 14 (App Router)
- Tailwind CSS v3
- Vercel hosting
- Stripe Billing (subscriptions)
